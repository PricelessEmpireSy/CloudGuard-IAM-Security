# CloudGuard - Remediation \& Security Controls

|**CONTROL AREA**|**ISSUE ADDRESSED**|**CONTROL IMPLEMENTED**|
|-|-|-|
|IAM|No MFA for some roles|Enforce MFA for all users|
|Access Control|Excess privileges|Apply least privilege|
|Storage Security|Public S3 buckets|Block public access by default|
|Monitoring|No real-time alerts|Enable cloud security monitoring|
|Compliance|No periodic reviews|Schedule quarterly access audits|

## Framework Mapping

* NIST CSF: PR.AC, DE.CM, RS.MI
* CIS Cloud Controls
* ISO 27001: A.9, A.12, A.16
